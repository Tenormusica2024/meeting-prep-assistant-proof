# Showcase Copy

## Short version

Meeting Prep Assistant Proof is a sample-first AI secretary demo that turns meeting inputs into draft-only preparation briefs. It checks agenda, context, open questions, and external-action needs without sending anything to participants or editing calendars.

## Portfolio-style version

This proof demonstrates an AI-secretary workflow for safer pre-meeting preparation. Synthetic meeting records are classified into ready, needs-agenda, needs-context, needs-confirmation, and send-later groups. The assistant creates draft-only briefs and moves participant-send or calendar-update suggestions into a human confirmation queue.

The value is not autonomous meeting operations. The value is the safer operating pattern: check readiness first, explain what is missing, prepare a brief, and require confirmation before external side effects.

## Safety notes

- sample data only
- no real participant names or meeting links
- no participant send
- no calendar update
- no OAuth requirement for the default demo
- confirmation queue for action-worthy meeting operations
